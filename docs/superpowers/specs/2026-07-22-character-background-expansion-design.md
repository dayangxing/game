# 角色背景内容池扩充设计

## 目标

将正式角色创建的背景内容池扩充为大型版本：18 项出身、18 项灵根、24 项命格天赋。保持现有 seed 随机、预览一致性和角色数据结构不变。

## 内容边界

- 出身覆盖山野平民、世家旁支、宗门经历、商旅医药、军伍和特殊遗民等来源。
- 灵根覆盖单灵根、双灵根、异灵根、变异灵根和少量稀有灵根。
- 命格天赋同时包含正向、负向和混合型特征，避免所有角色只获得正面词条。
- 每个角色仍随机获得 2～3 项命格天赋。
- 本次只扩充字符串池，不给新条目添加数值效果；现有事件和 LLM 仍把它们作为角色背景事实使用。

## 兼容性

`rollCharacter`、预览接口和正式创建继续使用同一套数组与 seed 逻辑。已有存档已经保存具体字符串，不会因代码更新而重新抽取；新角色使用相同 seed 时，由于池子扩大，可能得到不同背景，这是预期的内容扩充行为。

## 验证

- 导出角色池容量测试：18/18/24，且每个池内部无重复。
- seed 相同的角色仍完全相同，traits 数量仍为 2～3。
- 不同 seed 仍能产生不同背景组合。
- 后端角色创建、预览、前端角色创建和 storyDirector prompt 回归测试通过。
